# Flask Bloglite (Simple blog post website)

## Setup & Installation

Just need a latest version of Python installed.

```bash
pip install -r requirements.txt
```

## Running the App

```bash
python main.py
```

## Viewing the app

Go to `http://127.0.0.1:5000`
